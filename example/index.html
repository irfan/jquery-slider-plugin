jQuery Slider Plugin

Please check full documentation for other options of this plugin

How to use

Include the jquery, slider.js and slider.css files to your page

    <script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.3/jquery.min.js"></script>
    <script type="text/javascript" src="../slider.js"></script>
    <link rel="stylesheet" href="../slider.css" type="text/css" media="screen" />

Example HTML

<div id="sliderContainer">
    <ul>
        <li><img src="img/turkey.jpg"></img></li>
        <li><img src="img/going_to_base_camp.jpg"></img></li>
        <li><img src="img/climbing.jpg"></img></li>
        <li><img src="img/mountain.jpg"></img></li>
        <li><img src="img/walking.jpg"></img></li>
    </ul>
</div>

Basic Usage

<script>
    $(function(){
        $('#sliderContainer').slider();
    });
</script>

Default Options

    {
        method: 'circle',   // flat or circle
        auto: true,         // if set up as false you have to start slider by trigger start.slider event. 
        wait: 3000,         // plugin have to wait before go to next slide
        effect: 'slide',    // just support the slide
        duration: 300,      // effect duration
    }

Example of user defined options

<script>
    $(function(){
        $('#sliderContainer').slider({
            wait:2500,
            method:'flat'
        });
    });
</script>
